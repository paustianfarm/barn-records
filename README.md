# Barn Records

A mobile-friendly, single-file HTML web application for recording livestock (pig) health events at Paustian Enterprises.

## Features

- **Mortality Records** -- Log pig deaths with date, location, count, and weight.
- **Treatment Records** -- Track medical treatments with medication type, dosage calculations, caretaker, and withdrawal periods.
- **Auto Dosage Calculation** -- Automatically calculates medication dosages based on pig weight.
- **Google Sheets Sync** -- Syncs data to Google Sheets via Google Apps Script.
- **Offline Support** -- Works offline using localStorage.

## Tech Stack

Vanilla JavaScript in a single `index.html` file. No build tools or dependencies. Uses Google Sheets as a backend via Apps Script.

## Getting Started

Open `index.html` in a web browser. No build process required.
